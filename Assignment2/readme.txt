## HTML Tags Used

- `<header>`: Top navigation and branding.
- `<nav>`: Main navigation links.
- `<main>`: Main content wrapper.
- `<section>`: Major content areas (hero, experience, projects, gallery, testimonials, contact).
- `<article>`: Individual experience and project cards.
- `<aside>`: Sidebar info in about/contact.
- `<footer>`: Copyright and social links.
- `<table>`: Skills and certifications.
- `<form>`: Contact form.
- `<button>`: Call-to-action and form submission.
- `<img>`: Profile, project, and gallery images.
- `<ul>`, `<li>`: Navigation and lists.

## CSS Selectors

- `.info-table` and `.info-table tbody tr:nth-child(even)`: For table styling.
- `.nav-link:hover`, `.gallery-item:hover`, `.testimonial:hover`: For interactive effects.
- Flexbox and grid layouts for responsive design.
