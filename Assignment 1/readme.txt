# AI for Youth – Learn, Grow, Innovate

This website introduces youth to Artificial Intelligence, useful tools, and career opportunities.

## Tags Used
- Favicon → `<link rel="icon">`
- Semantic elements → `<header> <section> <article> <aside> <footer>`
- Table → AI tools comparison
- Form → Text, email, password, datalist, submit button
- Images → `<figure>` with `<figcaption>`
- Hyperlinks → Navigation + contact info
- Button → Form submit
- Audio → AI audio clip
- Video → AI careers video
- Details & Summary → Fun AI fact
- tel & mailto → Contact section

## Purpose
To empower youth with knowledge about AI in education, skills, and careers.
